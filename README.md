
# Foji Investment

Foji Investment is a stock investment tracking project designed to monitor market prices and manage investment portfolios.  
The goal of this project is to provide a simple structure for handling investment data, portfolio tracking, and profit/loss analysis.

## Features
- Portfolio tracking
- Market price monitoring
- Profit and loss calculation
- Simple and clean project structure

## Technologies
- JavaScript
- Node.js
- API Integration (optional)

## Installation

1. Clone the repository

git clone https://github.com/your-username/foji-investment.git

2. Go to the project folder

cd foji-investment

3. Install dependencies

npm install

4. Run the project

npm start
